# Future Academy Link — Desktop GUI

Electron desktop panel for **Future Academy Link** (`scratch-devices-link-lib`).

## Build

```bash
npm run build:gui:win
npm run build:gui:mac:arm64
npm run build:gui:mac:x64
```

## Outputs

- Windows: `dist/electron/win-unpacked/Future Academy Link.exe`
- macOS ARM64: `dist/electron/mac-arm64-unpacked/Future Academy Link.app`
- macOS Intel: `dist/electron/mac-unpacked/Future Academy Link.app`
